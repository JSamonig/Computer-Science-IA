To run:
1) set environment variable FLASK_APP=run.py
2) run the command: flask run
3) open localhost:5000

To debug:
1) set environment variable FLASK_APP=run.py
2) set environment variable FLASK_ENV=development
3) run the command: flask run
4) open localhost:5000
